Flexbox ‚û°Ô∏è

Flexbox is a one-dimensional layout method for laying out items in rows or columns. Items flex to fill additional space and shrink to fit into smaller spaces.

Flexbox is a whole module and not a single property.

Flexbox is direction-agnostic as opposed to the regular layouts (block which is vertically-based and inline which is horizontally-based). While those work well for pages, they lack flexibility (no pun intended) to support large or complex applications (especially when it comes to orientation changing, resizing, stretching, shrinking, etc.).

Flex container is the parent element to the flex items. It is also known as a flexbox. It enables a flex context for all its direct children.

Flex items are the children of a flex container. Flex items are the direct children of a flex container and are laid out inside the flex container‚Äôs main or cross axis, depending on their flex-direction.

Flex-direction ‚û°Ô∏è

The flex-direction property establishes the main-axis (default: row) and the direction (normal or reversed).

row (default): left to right in ltr; right to left in rtl
row-reverse: right to left in ltr; left to right in rtl
column: same as row but top to bottom
column-reverse: same as row-reverse but bottom to top

Positioning items in a flex container ‚û°Ô∏è

justify-content aligns items along the main axis.

flex-start (default): items are packed toward the start of the flex-direction.
flex-end: items are packed toward the end of the flex-direction.
start: items are packed toward the start of the writing-mode direction.
end: items are packed toward the end of the writing-mode direction.
left: items are packed toward left edge of the container, unless that doesn‚Äôt make sense with the flex-direction, then it behaves like start.
right: items are packed toward right edge of the container, unless that doesn‚Äôt make sense with the flex-direction, then it behaves like start.
center: items are centered along the line
space-between: items are evenly distributed in the line; first item is on the start line, last item on the end line
space-around: items are evenly distributed in the line with equal space around them. Note that visually the spaces aren‚Äôt equal, since all the items have equal space on both sides. The first item will have one unit of space against the container edge, but two units of space between the next item because that next item has its own spacing that applies.
space-evenly: items are distributed so that the spacing between any two items (and the space to the edges) is equal.
stretch (default): stretch to fill the container (still respect min-width/max-width)

align-items aligns items along the cross axis.

stretch (default): stretch to fill the container (still respect min-height/max-height)
flex-start: items are packed toward the start of the flex-direction.
flex-end: items are packed toward the end of the flex-direction.
start: items are packed toward the start of the writing-mode direction.
end: items are packed toward the end of the writing-mode direction.
center: items are centered along the line
baseline: items are aligned such as their baselines align

Responsive flexbox properties ‚û°Ô∏è

flex-wrap controls whether flex items are forced into a single line or can be wrapped onto multiple lines. If wrapping is allowed, it controls the direction that lines are stacked.

nowrap (default): all flex items will be on one line
wrap: flex items will wrap onto multiple lines, from top to bottom.
wrap-reverse: flex items will wrap onto multiple lines from bottom to top.

Grid ‚û°Ô∏è

CSS Grid is a two-dimensional layout system for the web. The grid is the intersecting set of horizontal and vertical grid lines formed by placing grid lines at the start and end of each column and row.

Layout ‚û°Ô∏è

Grid container is the parent element to the grid items. It is also known as a grid box. It enables a grid context for all its direct children.
To declare a grid container, add display: grid to the element.

Grid cells are cells inside a grid container. Is the smallest unit on a grid. Conceptually it is like a table cell. When the grid is defined as a parent the child items will lay themselves out in one cell each of the defined grid.

Grid-template-columns & Grid-template-rows‚û°Ô∏è

The grid-template-columns property defines the columns of the grid with a space-separated list of values. The grid-template-rows property defines the rows of the grid with a space-separated list of values. The values represent the track size, and the space between them represents the grid line.

auto: the size of the column is the size of the content
1fr: the size of the column is the remaining space after all other columns are laid out
repeat: repeat the pattern (repeat(3, 100px))
minmax: minimum and maximum size of the column (minmax(100px, 400px))

To move a item to a specific row and column we can use the grid-column and grid-row properties.

If we want to move div `Three` to 2nd row and 3rd column we can use the following properties:

grid-column-start: 3
grid-column-end: 4
grid-row-start: 2
grid-row-end: 3


Alternatively, we can use: grid-column: 3 / 4; grid-row: 2 / 3; which is a shorthand for the above properties.
An item can span multiple rows and columns.

Grid-gap, grid-column-gap, grid-row-gap‚û°Ô∏è

We can add a gap between the columns and rows using:

grid-column-gap: 10px
grid-row-gap: 10px

This will add a gap of 10px between the columns and rows.

Alternatively, we can use the grid-gap property.
The grid-gap property is a shorthand property for the grid-row-gap and the grid-column-gap properties.
